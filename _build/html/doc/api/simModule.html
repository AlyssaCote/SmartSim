SmartSimModule¶

class smartsim.simModule.SmartSimModule(state, **kwargs)[source]¶

The base class of all the modules within SmartSim. The SmartSim modules as it stands today are:

  • Generator

  • Controller

Each of the SmartSimModule children have access to the State information through this class.
