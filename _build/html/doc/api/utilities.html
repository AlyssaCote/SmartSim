Utilities¶

A set of utilities designed to make creating, using, and debugging Smart Sim experiments.

smartsim.utils.log.get_logger(name=None, log_level=None)[source]¶

Returns a log handle that has had the appropriate style set to ensure logging practices are consistent and clean across the code base.

Parameters

  • name (str) – the name of the desired logger.

  • log_level (int) – what level to set the logger at. Valid values are defined in the python logging module.

smartsim.utils.log.log_to_file(filename, log_level=None)[source]¶

Installs a second filestream handler to the root logger, allowing subsequent logging calls to be sent to filename.

Parameters

  • filename (str) – the name of the desired log file.

  • log_level (int) – as defiend in get_logger. Can be specified to allow the file to store more or less verbose logging information.

smartsim.utils.log.set_debug_mode()[source]¶

Sets the root logger to debug mode, and clears the logging level set on other loggers such that they log at the debug level.

smartsim.utils.log.set_global_logging_level(log_level)[source]¶

Sets the root logging level, which subsequent loggers inherit from unless setLevel has been called on them.

Parameters

log_level (int) – as defined in get_logger.

smartsim.utils.log.set_logging_level(logger, log_level)[source]¶

Sets the logging level for an individual logger. Typically, a logger inherits the log_level from root (and is NOTSET).

Parameters

  • logger – the handle to the logger.

  • log_level (int) – as defined in get_logger.
