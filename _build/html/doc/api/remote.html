Remote¶

smartsim.remote.remote.init_command_server(addr='127.0.0.1', port=5555)[source]¶
Initialize a command server so that SmartSim can be

run on compute nodes where slurm and other workload manager commands are not available. Command servers must be located on the same system, and be reachable via tcp.

Keep in mind that this does not actually start the CmdServer instance, that must be started manually.

Parameters

  • addr (str, optional) – address of the Command Server launched seperately, defaults to “127.0.0.1”

  • port (int, optional) – port of the Command Server, defaults to 5555

Raises

CommandServerError – If setup with remote launcher fails

smartsim.remote.remote.shutdown_command_server()[source]¶
Shutdown a command server launched by the user by

sending a shutdown request to the running CmdServer.

Raises

CommandServerError – If shutdown fails
